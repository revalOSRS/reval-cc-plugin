# Reval Clan

A RuneLite plugin for managing and enhancing the Reval clan experience in Old School RuneScape.

## Features

- Clan member management
- Custom welcome messages
- Clan chat utilities

## Installation

This plugin is available through the RuneLite Plugin Hub. To install:

1. Open the RuneLite client
2. Click on the wrench icon (Configuration)
3. Click on the Plugin Hub button (puzzle piece icon)
4. Search for "Reval Clan"
5. Click Install

## Configuration

The plugin can be configured in the RuneLite settings panel:

- **Welcome Greeting**: Customize the message shown when you log in

## Usage

Once installed and enabled, the plugin will automatically enhance your clan experience.
